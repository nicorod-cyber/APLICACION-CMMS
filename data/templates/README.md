# Plantillas HTML

Aqui se versionaran plantillas HTML configurables para PDF, correos y respaldos documentales.

Plantillas actuales:

- `alert-default.html`: plantilla base para alertas, correos y PDF de notificacion.

Las variables se reemplazan con sintaxis `{{NombreCampo}}`, por ejemplo `{{Title}}`, `{{Message}}`, `{{Severity}}`, `{{Source}}`, `{{EntityId}}` y `{{FaenaCodigo}}`.
