# Importaciones Excel

Este directorio almacena el repositorio controlado de importaciones.

- `originals/`: archivos Excel cargados por usuarios.
- `metadata/`: previews, errores, resumen, estado de aprobacion y trazabilidad.

Las importaciones no actualizan maestros oficiales hasta aprobarse desde la API o la pantalla de importaciones.

La documentacion funcional esta en `docs/excel-imports.md`.
